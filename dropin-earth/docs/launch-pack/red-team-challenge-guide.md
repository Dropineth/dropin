# Red-Team Challenge Guide

## What Can Be Challenged

- Campaign claims.
- Lottery rounds.
- Randomness certificates.
- Payment Intents.
- Reconciliation anomalies.
- Evidence objects.
- Impact Certificates.
- Fund allocations.
- Settlement certificates.
- Solana root anchors.

## Evidence Examples

- Duplicate transaction hash.
- Wrong recipient, amount, memo, or network.
- Missing testnet-only disclosure.
- Wording that implies certified carbon credit status.
- Wording that implies guaranteed yield.
- Wording that implies $CANOPY offsets carbon tax.
- Reused photo, wrong GPS, or suspicious evidence.
- Public report mismatch.

## Process

```text
Submit Challenge
→ Attach Evidence
→ Admin / Red Team Review
→ Target marked challenged if accepted
→ Resolution audit log
→ Protocol or copy fix
```

## Recognition

Valid testnet challenges may receive placeholder Leaf Points. Leaf Points are
non-transferable testnet points only.
